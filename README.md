# LibAFL 文档书

这个项目作为一本书，包含了 LibAFL 源码以外的文档。

在这里你可以找到教程、例子和详细的解释。

要想获得API文档，请在 LibAFl 根目录下运行 `cargo doc`。

## 使用方法

要构建这本书，你需要 [mdBook](https://github.com/rust-lang/mdBook)。

`mdbook build` 来构建，`mdbook serve` 来在本地提供本书。
